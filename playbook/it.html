08 · IT

IT и системы

Технологический стек сети: Matchpoint, касса, CRM, внутренний портал V7, Racket.ID, интеграции и доступы.

Статус: Черновик
В разработке

Часть информации уже есть в V7 Brain (технический источник правды для данных и архитектуры). Здесь — операционный взгляд для управляющих.

Структура блока

  1. Matchpoint — CRM клуба, бронирования, клиенты, базовая кассовая функция.
  2. Касса — фискальная дисциплина, локальные требования по клубу.
  3. Внутренний портал V7 — dashboard, management, finance. Доступы по ролям.
  4. Racket.ID — рейтинги, турниры, участники.
  5. Google Sheets — P&L, финучёт бухгалтера.
  6. Онлайн-каналы — сайт, Instagram, Telegram.
  7. Роли и доступы — admin / manager / coach / administrator.
  8. Онбординг IT для нового клуба — что подключается при открытии и в каком порядке.

Роли доступа в портале (из auth-guard)

  • admin — CEO сети, видит всё, включая финансы и стратегию.
  • manager — управляющий клуба, видит клиентов, членов, турниры, recall, дашборды.
  • coach — тренер, видит клиентов, турниры, расписание, калькулятор.
  • administrator — администратор ресепшн, видит клиентов, турниры, расписание.
