# 20 · Какие доступы нужны Claude (для не-айтишника)

> Список того что нужно открыть мне (Claude) чтобы я делал работу сам, без участия Макса/Эрдема. Каждый пункт — короткое объяснение «что это», «зачем нужно», «как дать».

## 🟢 СРОЧНО (нужно для работы этой недели)

### 1. Webflow — доступ к сайту

**Что**: Платформа на которой сделан v7padel.com. Без доступа я не могу править SEO, добавить трекинги, фиксы дизайна.

**Зачем**: Все 6 фиксов сайта (meta description, Open Graph, Schema.org, hreflang, cookie banner, email capture) — делаются через Webflow.

**Как дать**:
1. Зайти на webflow.com → войти как owner
2. Site Settings → **General** → раздел **Members**
3. Кнопка **Invite member** → ввести email Claude (тот что я скажу ниже)
4. Роль: **Designer** (не Editor — мне нужны Custom Code права)
5. Нажать «Send invitation»

**Кому именно**: дай мне email-адрес куда отправлять — я создам новый Webflow-аккаунт под этот email.

---

### 2. Google Analytics 4 (GA4)

**Что**: Аналитика трафика на сайте. Сколько людей зашло, откуда, что смотрели, забронировали ли.

**Зачем**: Без этого Эрдем тратит на рекламу слепо — не знает что работает.

**Есть ли уже**: Скорее всего нет (сайт без трекинга). Надо создать.

**Как дать** (если есть):
1. analytics.google.com → выбрать аккаунт V7
2. **Admin** (шестерёнка слева внизу) → **Account access management**
3. Добавить email Claude → роль **Editor**

**Если ещё нет**: я создам бесплатно за 5 минут (нужен только Webflow доступ).

---

### 3. Meta Business Manager (Facebook + Instagram)

**Что**: Кабинет где Эрдем запускает рекламу FB/IG.

**Зачем**: Установить Meta Pixel на сайт = ретаргетинг, lookalike-аудитории, точное измерение конверсий.

**Как дать**:
1. business.facebook.com → войти как admin
2. **Business Settings** → **People** → **Add**
3. Добавить email Claude → роль **Admin** или **Marketing Analyst**

**Если нет Business Manager**: делаем — Эрдему 5 минут создать, ему всё равно нужно для рекламы.

---

### 4. Бэкап `sa.json` и `matchpoint_creds.json`

**Что**: Два файла-ключа от Google Sheets и от Matchpoint которые сейчас лежат **только** на твоём ноутбуке. Если ноут украдут — мы потеряем доступ ко **всем данным V7**.

**Зачем**: Disaster recovery. Это **критично**.

**Как дать**:
1. Создай папку в твоём личном Google Drive: «V7 Secure Backup»
2. Открой её только тебе (никому не шарь)
3. Скажи мне какой email — я положу туда копии файлов

**Альтернатива**: 1Password / Bitwarden vault — храним там. Я могу настроить если хочешь.

---

### 5. GitHub — доступ для AI Actions

**Что**: Репозиторий где живёт код v7padel.

**Зачем**: Перенести cron на GitHub Actions (надёжность 100%, не зависит от твоего ноута).

**Как дать**:
1. github.com → omalkut-prog/v7padel → **Settings** → **Collaborators**
2. Add people → ввести github username который я скажу
3. Роль: Write

**Я уже работаю** через MCP с твоим репо, но для GitHub Actions нужен именно collaborator-доступ + один Personal Access Token (10 минут настройки).
