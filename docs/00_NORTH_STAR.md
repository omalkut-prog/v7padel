# 00 — North Star & Strategy

## North Star Metric

**80 000 ₺/день выручки (скользящая средняя 30 дней)**

Считается по всем источникам: тикеты + saldo-движения + bono-движения.
Live-значение: см. `04_CURRENT_STATE.md` → секция Finance.

Почему именно это: один скалярный KPI, устойчивый к дневным флуктуациям, отражает здоровье бизнеса целиком (не только загрузка, не только клиенты).

---

## Стратегия 4 контуров

Четыре источника выручки, каждый с своей динамикой:

| Контур | Суть | Основная метрика |
|---|---|---|
| **1. Аренда кортов** | Почасовое бронирование | Заполняемость % · ₺/корто-час |
| **2. Абонементы (Club/VIP)** | MRR-подобный доход | MRR ₺ · churn % |
| **3. Тренировки и академия** | Пакеты с тренерами | ₺ с пакета · retention |
| **4. Бар + магазин** | Доп. выручка со визита | ATV (средний чек) · attach rate |

Цель — балансированный рост по всем четырём, не перекос в один.

## Позиционные правила (неизменяемые)

Эти правила не пересматриваются квартально — это ДНК бизнеса.

1. **Премиум-тир**. Качество кортов + тренерский состав + сервис + комьюнити.
2. **НЕТ скидок**. Никаких «−20%», «2 по цене 1», промокодов, flash sale.
3. **НЕТ distressed-акций**. «Последний шанс», «горит слот», «только сегодня» — это не наш тон.
4. **Цена — часть продукта**. Если клиент ищет дешевле — он не наш клиент.
5. **Ценообразование — это решение, а не переговоры**. Прайс-лист публикуется, ни с кем не торгуемся.

## Non-goals (что мы НЕ делаем)

- Не снижаем цены чтоб увеличить поток
- Не копируем механики массовых фитнес-клубов
- Не тратим деньги на перформанс-маркетинг пока не пройдена точка юнит-экономики
- Не расширяемся на другие локации до выхода на стабильные 80 k₺/день
