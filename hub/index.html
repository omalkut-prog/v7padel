🤖 HubV7 — контекст V7 Padel

Одна ссылка → HubV7 знает всё про твой клуб

Способ 1 — Дать HubV7 ссылку

Если HubV7 умеет читать URL, просто отправь ему:

https://v7padel.space/hub/v7-context.md

С сообщением:

Прочитай мой контекст по этой ссылке и работай по этим правилам: https://v7padel.space/hub/v7-context.md
Открыть файл

Способ 2 — Скопировать всё в первое сообщение

Если HubV7 не читает URL — скопируй весь файл и вставь в первое сообщение HubV7:

  1. Открыть файл v7-context.md
  2. Ctrl+A → Ctrl+C
  3. В HubV7: "Вот мой контекст по V7 Padel, запомни и работай по этим правилам:" + Ctrl+V

Способ 3 — Project Knowledge

Если HubV7 поддерживает Projects (как Claude Projects):

  1. Создай Project «V7 Padel Antalya»
  2. В Custom Instructions вставь содержимое 00_SYSTEM_PROMPT.md
  3. В Project Knowledge загрузи v7-context.md
