# V7 Padel — Entry point для ИИ

> **Первый раз в проекте?** Прочти в порядке: `docs/04_CURRENT_STATE.md` → `docs/05_BACKLOG.md` → `docs/00_NORTH_STAR.md`. Получишь 80% контекста за 5 минут.

## Что это за проект

Клуб падела в Анталье (V7 Padel Antalya), 4 корта, 781 клиент в базе. Собственник — Володимир. Строим аналитический портал поверх Matchpoint (CRM клуба) + Google Sheets, чтоб управлять клиентами, выручкой, загрузкой, тренировками.

- **Язык**: все ответы и документы на русском. Без английских вставок.
- **North Star**: 80 000 ₺/день выручки (скользящая 30 дней).
- **Позиционирование**: премиум. Скидок нет. Конкурируем качеством.

## Update cadence (обязательно)

### Авто (без человека)
- `04_CURRENT_STATE.md` — каждый ETL run (`etl/generate_state_doc.py`)

### В каждом PR (ИИ обязан)
- Меняется ETL-скрипт → обнови `03_ETL_PLAYBOOK.md` или `02_MATCHPOINT_API.md`
- Меняется схема таблиц → `01_ARCHITECTURE.md`
- Меняются цены/часы → `10_PRICES_AND_HOURS.md`
- Неочевидное решение → `09_DECISIONS.md` (5 строк: контекст, варианты, выбор, почему)
- Факап → `07_ANTI_PATTERNS.md` (1 строка с ссылкой на коммит)

Если doc не меняется — в коммит-сообщении флаг `NO_DOC: причина`.

### Еженедельно (воскресенье 18:00, 30 мин, Володимир + ИИ)
- Пересмотр `05_BACKLOG.md`: что сделано → вычеркнуть, NOW → пересмотреть
- Быстрый скан `04_CURRENT_STATE.md`: есть ли разрыв план/факт

### Ежемесячно (1-е число, 60 мин)
- Ревью `11_MARKETING.md`
- Проверка `00_NORTH_STAR.md`: KPI-цели реалистичны?
- Актуализация `08_TEAM_AND_ROLES.md`

### Ежеквартально (1-е число квартала, 90 мин)
- `00_NORTH_STAR.md`: топ-3 цели на новый квартал
- `06_PRINCIPLES.md`: добавить/убрать что-то
- Генеральный аудит устаревшего

## Критические правила

1. **Никаких скидок**. Премиум-позиционирование. Если спрашивают — подумай как сказать «нет», а не «сколько».
2. **Numbers только из `04_CURRENT_STATE.md`**. Не копируй цифры в другие доки — ссылайся.
3. **Max 5 в `NOW`**. Шестой пункт не попадает пока пятый не закрыт.
4. **ADR на каждое неочевидное решение**. 5 строк минимум, в `09_DECISIONS.md`.
5. **Русский язык, женский/мужской — по контексту**. Володимир, Оля (PM) — мужчина/женщина соответственно.
6. **Оля — PM (Project Manager)**, правая рука Володимира. Не CEO.
7. **Rent (380 490 ₺/мес) в `04.41 PnL`**, `fixed_opex.json` — только для live-месяца. Не для прошлых.
8. **Staff исключать из клиентских отчётов**: cid 14 (Olha Bo), 5, 7, 9, все @v7padel.com.
